# Security Policy

Polykube is in private alpha. Security reports should be sent privately to the repository maintainers until a public disclosure process is established.

## Supported Versions

No public supported versions exist yet.

## Reporting a Vulnerability

Open a private security advisory in GitHub or contact the maintainers directly through the owning organization.

Please include:

- affected component
- reproduction steps
- expected impact
- suggested mitigation, if known

Do not include live credentials, cloud account identifiers, or production secrets in reports.
